# Project posters

This folder contains final project-poster assets used by portfolio project cards.

- Recommended format: PNG or WebP.
- Recommended aspect ratio: vertical 3:4 or 2:3.
- Recommended resolution: at least 1200px on the long edge.
- Initial expected QueryOps file name: `queryops-product-poster.png`.

During development, the portfolio poster preview falls back to a designed placeholder when a poster asset is unavailable.
